IO
===
使用 [Google android-serialport-api](https://code.google.com/p/android-serialport-api) JNI代码编译的串口库类
以及利用 Linux 内核用户态的功能监听 GPIO 引脚

使用说明
---
```
implementation 'cn.6xyun.android:io:x.y.z'
```
